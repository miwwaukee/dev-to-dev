# DevHelper MCP Server 🛠️

MCP-сервер с инструментами для ежедневной работы разработчика.

## 🎯 Что решает

| Проблема | Решение |
|----------|---------|
| Технический долг | Поиск TODO/FIXME с приоритизацией |
| Уязвимые зависимости | Локальная проверка без внешних API |

**Для кого полезен:** Разработчики, тимлиды, DevOps перед релизом.

**Реализовано 2 инструмента:**
1. `find_tech_debt` — сканирует код на техдолг
2. `check_dependencies` — проверяет зависимости на уязвимости

---

## 🚀 Быстрый старт

### Требования
- Docker установлен и запущен
- Порт 8000 свободен

### 1. Сборка образа
```bash
docker build -t devhelper-mcp .
### 2. Запуск сервера
```bash
docker run -p 8000:8000 devhelper-mcp serve
